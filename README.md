# Vote App

Application de vote collaboratif en temps réel.

## Lancer en local

```bash
npm install
npm start
```

Ouvrir http://localhost:3000

## Déployer sur Render.com

1. Créer un compte sur https://render.com
2. Connecter ton repo GitHub
3. Créer un "New Web Service"
4. Sélectionner le repo
5. Configuration :
   - **Build Command** : `npm install`
   - **Start Command** : `npm start`
6. Cliquer "Create Web Service"

L'app sera accessible sur une URL du type `https://vote-app-xxxx.onrender.com`
